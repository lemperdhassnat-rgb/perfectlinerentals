Description

A responsive car rental website built with HTML, CSS, and JavaScript.
Shows available cars, daily rental prices in AED, and allows users to book cars directly.

Features
Displays cars with images, name, type, specs, and price (AED/day)
Badges for popular/new cars
Contact info localized to Dubai, UAE
Simple booking form with +971 phone input
Easy to customize and add more cars
Installation
Clone or download the repository.
Open index.html in your browser.
No server required — pure frontend demo.
How to Use
Browse available cars on the main page.
Check specs (automatic, petrol/hybrid, seats).
Click “Rent Now” to trigger booking.
Call the displayed UAE phone number +971 54 495 7070 for confirmation.
Customization
Add new cars: Update the cars array in the JS section.
Change currency: Modify AED in the price display.
Update contact: Change phone number in HTML input and contact sections.
